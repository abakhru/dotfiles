CTF workshop flow:

- 1 mins overview of CTF slides.
- Send them a wiki page with instructions to setup their environment to run CTF for ESA,
in advance so they have the environment ready for the workshop.
- Creating a test case from scratch, run it and show result.
- Then, give each of the attendees a simple test case to create, run and show results (answer any questions they may have).
